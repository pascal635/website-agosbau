# Onboarding-Assets

Ablage fuer alle Kunden-Assets, die fuer die Website gebraucht werden.

## logo/
Logo-Varianten des Kunden.
- hell (fuer dunkle Hintergruende)
- dunkel (fuer helle Hintergruende)
- icon (quadratisch, ohne Schriftzug)
- favicon
Formate: SVG (bevorzugt), zusaetzlich PNG mit transparentem Hintergrund. Favicon als PNG 512x512 oder ICO.

## ci/
Corporate Identity / Markenrichtlinien.
- CI-Manual / Styleguide (PDF)
- Farben (HEX/RGB, als Liste oder im Manual)
- Schriften (Font-Dateien WOFF2/TTF/OTF oder Namen + Quelle)

## images/
Bildmaterial vom Kunden.
- Betrieb (Gebaeude, Raeume, Standort)
- Team (Portraits, Gruppenfotos)
- Arbeiten (Projekte, Produkte, Referenzen)
Formate: JPG oder PNG, hohe Aufloesung (mind. 1920px breit), Originale unbearbeitet.
